# GetEgged

GetEgged allows players to capture mobs into spawn eggs while preserving important data such as name, age, taming, colors, and horse stats.

## Usage
Right-click a mob with the configurable capture item to store it in a spawn egg. use command /getegged get for capture item.

## Permissions
- getegged.animal.\<MOB>
- getegged.animal.*
- getegged.monster.\<MOB>
- getegged.monster.*
- getegged.golem.\<MOB>
- getegged.golem.*
- getegged.villager
- getegged.get

## Known Limitations
- Some mob variants (villager profession/type, cosmetic variants) are not yet restored.
