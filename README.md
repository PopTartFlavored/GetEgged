# GetEgged

GetEgged allows players to capture mobs into spawn eggs while preserving important data such as name, age, taming, colors, and horse stats.

## Usage
Right-click a mob with the capture item (stick for now) to store it in a spawn egg.

## Permissions
- getegged.animal.<MOB>
- getegged.animal.*
- getegged.monster.<MOB>
- getegged.villager

## Known Limitations
- Some mob variants (villager profession/type, cosmetic variants) are not yet restored.
